================================================================================
TROUBLESHOOTING üîß
================================================================================

PROBLEM: "Please add your Stripe API key" error
SOLUTION: Replace YOUR_STRIPE_PUBLISHABLE_KEY_HERE in complete-site.html
          with your actual test key from Stripe dashboard

PROBLEM: Payment button does nothing
SOLUTION: Open browser console (F12) and check for errors
          Make sure local server is running

PROBLEM: Cart not updating or showing items
SOLUTION: Clear localStorage and refresh
          1. Open browser console (F12)
          2. Run: localStorage.clear();
          3. Run: location.reload();

PROBLEM: 3D bikes not showing
SOLUTION: You must run a local server (not file:// protocol)
          Use: npx http-server -p 8000 -c-1

PROBLEM: Cart is empty after page refresh
SOLUTION: This is normal on checkout success page
          Cart is cleared after successful payment
          On other pages, cart persists in localStorage

PROBLEM: Stripe Elements not loading
SOLUTION: Check internet connection
          Stripe.js loads from CDN and requires internet
